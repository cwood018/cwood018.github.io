Work

Image Processing Parking Enforcement

  • Capstone project built to reduce the cost of manpower associated with preventing unauthorized parking
  • Python OCR script to detect invalid parking passes via license plate querying on school grounds using OpenCV import
  • Script utilizes boto3 import to use AWS S3 for video storage and AWS RDS for MS SQL Express Database
  • User interface was created for enforcement officers using .NET framework to enable follow up on ticketing history of unauthorized and unidentified vehicles within the system
  • System provided evidence of ticketing with timestamp of ticket, as well as S3 location of frame with vehicle in question

storyteller

  • Python and React web application enabling users to upload documents to manipulate and visualize data
  • Utilizes many AWS modules such as S3 for user data and files, Lamdas for documentation upload and Textract for document extraction
  • Users can use their uploaded data from S3 to visualize their data and present to senior management and clients
  • Content can be extracted from a CSV, Excel Spreadsheet or extracted from PDF via AWS Textract

Stock Analysis

  • Python web scraping script to gather prices from Yahoo Finance Daily, this is a passion project and still in progress
  • Tables collected are Commodities, Currencies, Cryptocurrences, and Most Active Stocks
  • AWS Lambda is triggered daily to perform the scrape and insert into an AWS RDS table for analysis
  • A React application is in developement to present the data for the users to make investment decisions as well as visualizing the stocks over a period of time

HealthTrax

  • .Net Web Application to create a caloric counter for users to live a healthy lifestyle and track their BMI
  • The caloric intake was broken down based on macronutrient groups - protein, carbohydrates and fat
  • Another module was the sleep tracker, the user could enter their time of sleep and wake up to calculate amount of sleep per week
  • The user's BMI would update weekly to receive new caloric daily totals to reach their goals
